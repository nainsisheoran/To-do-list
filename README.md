# 📝 To-Do List Application (CLI-Based)

## 📌 Project Overview
The To-Do List Application is a command-line based task management system developed using Python.  
It allows users to efficiently manage their daily tasks by adding, viewing, deleting, and saving tasks.

The application uses file handling to store tasks permanently, ensuring that user data is retained even after the program is closed.

---

## 🎯 Objectives
- Build a simple and user-friendly task manager
- Implement CRUD operations (Create, Read, Delete)
- Store tasks persistently using file handling
- Provide a menu-driven interface for ease of use

---

## 🚀 Key Features
- ➕ Add new tasks with descriptions  
- 📄 View all saved tasks  
- ❌ Delete specific tasks  
- 💾 Save tasks to file (`task.txt`)  
- 🔁 Automatic loading of saved tasks on restart  
- 🧾 Simple and interactive CLI interface  

---
